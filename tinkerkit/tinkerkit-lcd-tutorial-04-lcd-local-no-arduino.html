TinkerKit LCD - Tutorial 04 - LCD Local (No Arduino)

The TinkerKit LCD module can also be used by itself, without Arduino or TinkerKit! shield. That‚Äôs because it mounts the same microcontroller as the Arduino Leonardo; thanks to its micro-usb connector, it can be plugged straight into your computer and programmed from the software just like a regular Arduino Leonardo.

If you didn‚Äôt do it already, download the TKLCD library. Once downloaded, place TKLCD in the libraries folder inside your sketchbook.

How to install the library:

  1. unzip the downloaded file
  2. move the TKLCD folder inside the ‚Äúlibraries‚Äù folder, usually located in Documents\Arduino\libraries (if it‚Äôs not there, just create it)
  3. close the Arduino software, if it‚Äôs open, then relaunch it
  4. to verify the installation, open the software and from the top menu click on sketch->import library. You should be able to see the TKLCD element in the list

In this tutorial we‚Äôre not using the serial port, for this reason we don‚Äôt have to upload the firmware. Let‚Äôs begin by connecting the LCD module with a micro USB cable, then open the Arduino software. In top menu, under Tools->Board select Arduino Leonardo.

First of all we have to include all the libraries required by the LCD module: TKLCD, LiquidCrystal and Wire. The last two don‚Äôt need to be downloaded, they‚Äôre included in the Arduino software and are selectable just like the other libraries from Sketch->Import Library.

Now we have to create a LCD object in the globals, like every other module. There are two different classes for the TinkerKit! module, we have to declare it as TKLCD_Serial or TKLCD_Local according if we are using it via serial or via USB.

Then we have to initialize it in the setup():

Let‚Äôs start by writing something on our LCD screen: inside the setup() write

Every time that we want to print something new on the screen, we have to ‚Äúrefresh‚Äù it with the clear() function. To write ‚ÄúHello World!‚Äù but with the two words alternating every second on the screen, we have to type:

lcd.clear();
lcd.print("World!");
delay(1000);

Load it into the module and you should see "Hello" and "World" fleshing.

Now let‚Äôs use it to read the value of an input. Connect a thermistor to one of the connectors, then we‚Äôll use the LCD screen to read its values. Don‚Äôt forget to include the normal TinkerKit! Our loop function is now:

//print the values on the lcd
