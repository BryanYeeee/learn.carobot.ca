TinkerKit LCD - Tutorial 05 - Arduino & 2 Wires

The TinkerKit LCD module can be controlled straight from the Arduino, using the serial communication, without any shield or other module. Remember to upload the SerialFirmware into the LCD board, as we did in the four-cable serial tutorial. It‚Äôs located under examples->TKLCD->SerialFirmware

Example Code‚Äã

Upload to the Arduino any sketch that uses the TKLCD_Serial and reset the Arduino manually by pressing the reset button after the upload is completed. In the following example we‚Äôre just writing ‚ÄúHello World‚Äù on the board.

#include <matrix_lcd_commands.h>
#include <TKLCD.h>
 
TKLCD_Serial lcd = TKLCD_Serial();  
 
void setup() {
  lcd.begin();  
}
 
void loop() {
  lcd.clear();
  lcd.print("Hello");
  delay(1000);
   
  lcd.clear();
  lcd.print("World");
  delay(1000);
}
